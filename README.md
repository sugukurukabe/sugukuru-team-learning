# sugukuru-team-learning

**スグクル AI開発者育成プログラム v3.0**

Build with ADK, Equip with MCP, Communicate with A2A

## 原則
- IDEで全ての日常業務をやる
- 自動化できることは全て自動化する
- 今日からGitHubで仕事する

## ゴール
MCPサーバーを構築し、APIを繋げ、IDEと接続し、MCPツールを作れる一人前になる

                                                    ↓
                                            ★一人前認定★
